SciSuite
Science should be sweet.
SciSuite is a collection of 5 plugins for Adobe Photoshop and Illustrator, designed to help researchers prepare publication-ready scientific figures efficiently.

GuideMi — Guide & layout management
WesternBo — Western blot & gel annotation tools
ImageXi — Microscopy image processing
ChartEr — Chart annotation & tables
SciPanel — Figure panel assembly & batch export

Key Features

200+ specialized tools across 5 plugins
Vector-based output (shape layers, crisp at any resolution)
17-language localization
Single-step undo for every operation
Preset system for consistent lab-wide figure styles
