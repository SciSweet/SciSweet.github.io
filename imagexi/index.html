ImageXi

Microscopy Image Processing
PsAi

ImageXi is a Photoshop/Illustrator CEP plugin designed for scientific image processing, particularly optimized for microscopy images. It provides a rich set of automation tools for layer adjustments, text annotations, scale bar creation, and channel operations.

Interface Overview

Key Features

Quick Actions Document merging, layer swapping, channel splitting, gap measurement, and guide management.
Scale Bar Scale bar creation supporting both simple mode and professional mode.
Inset Magnified inset creation tool for duplicating and scaling selected regions with optional strokes and connector lines.
Pseudo Color Apply pseudo-color LUTs to grayscale images for channel visualization.
Add Label Automatic sequential labeling (A, B, C or 1, 2, 3) for multiple layers with replace mode.
Resize Layers Batch scale layers to specified dimensions.
Adv Line Marker Advanced multi-row annotation tool with cell merging and automatic line splitting.
